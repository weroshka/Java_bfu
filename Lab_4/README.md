# Шахматы (консольная версия)
___

### Цель проекта:
Написать программу на Java для игрвы в шахматы на 2 человек.
Правила игры стандартные.
*Обозначения фигур:*
* Пешка - Pawn
* Конь - Knight (рыцарь)
* Слон - Bishop (епископ)
* Ладья - Rook (грач/утес/башня)
* Ферзь - Queen (королева)
* Король - King
